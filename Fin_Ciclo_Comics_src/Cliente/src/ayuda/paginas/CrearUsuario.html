Crear Usaurio

Esta ventana será donde se realizarán las acciones de creación de usuarios sobre la base de datos

    Principalmente tenemos 3 campos de textos:
  • Usuario: se refiere al nombre que tendrá el usuario
  • Contraseña: referente a la contraseña asociada con el usuario
  • Repetir Contraseña debe tener el mismo valor que el campo anterior

A mayores de esto, esta ventana tambieén consta de 3 botones

  • ...: al hacer click sobre este boton, se abrirá una ventana de búsqueda para seleccionar una foto que asociar al usuario. La imagen escogida se representará a la izquierda
  • Añadir: comprobará que los valores de los campos sean los correctos y añadirá el usuario a la base de datos si resultan ser correctos.
  • Cerrar: Se cerrará esta ventana sin realizar ninguna acción
