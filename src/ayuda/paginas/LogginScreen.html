Inicio de sesión

Esta es la ventana principal de la aplicación

Consta de dos menús.

  • idioma: en este menú se podrá cambiar el idioma entre entre español, inglés o gallego.
  • Ayuda: este menú abrirá esta ayuda

Bajando al cuerpo de la ventana, podemos apreciar un texto "Desconectado". Este texto mostrará el estado con el servidor

Los campos Puerto e IP se encuentrán desactivados, para cambiarlos basta con hacer click en el boton Cambiar debajo de estos campos

    Botones:
  • El boton Probar conexión comprobará si es posible la conexión con el dispositivo deseado
  • El boton Cambiar abrirá la ventana para cambiar la ip y el puerto al que conectar
  • El boton Desconectar cerrará la conexión con el dispositivo conectado, en caso de no haber ninguna conexión se mostrará por pantalla
  • El boton Iniciar sesión Comprobará que el usuario y contraseña existan en la base de datos. De ser así, iniciará sesión y sla ventana Info Usuario será abierta
  • El boton Crear usuario Comprobará la conexión, y, si es posible conectar, abrirá la ventana Crear Usuario
