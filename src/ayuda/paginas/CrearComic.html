Crear Comic

En esta ventana se llevará a cabo el porceso de inserción de comics en la base de datos.

Todos los campos deben ser rellenados a excepción del campo 'descripción'.

Los campos 'precio' y 'cantidad' son numéricos, y solamente acetarán valores numéricos

Al hacer click sobre la imagen de la parte superior izquierda, se abrirá una ventana de selección de imágenes para que el usuario escoja

El campo colección será un desplegable con todas las colecciones existentes en la base de datos

Tras cubrir todos los campos, el boton 'Añadir' insertará el comic en la base de datos.

Si por el contrario no se desea insertar el comic, el boton 'cerrar' te devolverá a la ventana anterior sin hacer ninguna operación
